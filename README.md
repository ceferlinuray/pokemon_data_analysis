# pokemon_data_analysis
Pokémon dataset analysis using Python (Pandas, NumPy, Seaborn). Includes data cleaning, statistical analysis, and visualization of Pokémon attributes.
This project analyzes a Pokémon dataset using Python to explore patterns, relationships, and insights from different attributes such as type, stats, and rarity.

The analysis is performed using Jupyter Notebook and focuses on statistical exploration and data visualization.

📊 Objectives

The main goals of this project are:

Identify the most common Pokémon types
Detect outliers in Pokémon stats
Analyze relationships between attributes (e.g., capture rate vs base stats)
Compare legendary vs non-legendary Pokémon
Find the strongest Pokémon based on attack and total stats
Explore correlations between different feature
